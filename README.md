# 五子棋游戏

一款使用 HTML5 Canvas 和 JavaScript 实现的经典策略棋盘游戏。两名玩家轮流在 15×15 的棋盘上放置棋子，目标是首先在水平、垂直或对角线上连成五个棋子。

## 功能特点

- **精美界面**: 现代设计，渐变背景和平滑动画
- **交互式画布**: 点击放置棋子，提供视觉反馈
- **游戏逻辑**: 完整的规则实现，包括胜利检测
- **响应式设计**: 适用于桌面和移动设备
- **视觉效果**:
  - 棋子放置动画
  - 获胜棋子高亮显示
  - 下一步棋的悬停预览
- **游戏控制**:
  - 重新开始游戏
  - 撤销上一步
  - 显示提示（简单AI建议）

## 游戏玩法

1. **目标**: 将你的五个棋子连成一行（水平、垂直或对角线）
2. **回合**: 黑棋先走，然后玩家轮流进行
3. **放置**: 点击任意空交叉点放置你的棋子
4. **获胜**: 游戏自动检测玩家何时获胜
5. **重新开始**: 使用“重新开始游戏”按钮开始新游戏

## 安装和运行

### 前置条件

- **Node.js**: 版本 14.0.0 或更高（支持 ES6 模块）

### 快速开始

1. **克隆或下载** 项目文件
2. **安装依赖**（用于代码格式化）:
   ```bash
   npm install
   ```
3. **启动本地服务器**:
   ```bash
   npm start
   ```
4. **打开浏览器** 并访问:
   ```
   http://localhost:8000/
   ```

### 替代方法

**使用 Python**（如果 Node.js 不可用）:

```bash

```

**直接打开文件**（由于模块的CORS限制，不推荐）:
直接在浏览器中打开 `index.html`（可能无法与ES6模块一起工作）。

## 技术细节

### 游戏架构

- **棋盘类** (`board.js`): 管理游戏状态、移动验证、胜利检测
- **游戏控制器** (`game.js`): 处理Canvas渲染、用户交互、动画
- **工具函数** (`utils.js`): 坐标转换、验证辅助函数

### 关键算法

**胜利检测**:

- 从最后放置的棋子检查8个方向
- 计算相同颜色的连续棋子数量
- 当计数 ≥ 5 时返回胜利

**坐标系统**:

- 屏幕坐标 ↔ 棋盘坐标转换
- 处理响应式画布大小调整

### 依赖项

- **运行时依赖**: 纯 HTML5、CSS3 和 JavaScript (ES6+)，无外部依赖
- **开发依赖**: Prettier 用于代码格式化（通过 `npm install` 安装）
- **Font Awesome**: UI元素的图标（从CDN加载）
- **无框架**: 轻量级且快速
